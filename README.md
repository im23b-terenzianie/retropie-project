# 🕹️ Raspberry Pi RetroPie Projekt

Dies ist mein persönliches Emulator-Projekt mit RetroPie auf dem Raspberry Pi 4.

## 📦 Inhalt

- Backup-Skripte
- Eigene ROM-Verwaltung
- Gamepad-Setup-Dokumentation
- Autostart-Konfiguration
